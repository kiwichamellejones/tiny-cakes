# Tiny Cakes Bakery

Tiny Cakes Bakery is a polished front-end storefront concept for a boutique dessert brand. The project was redesigned from an earlier course assignment into a portfolio-ready experience focused on elegant branding, interactive browsing, and premium visual presentation.

## Overview

This project showcases a bakery website built with only HTML, CSS, and JavaScript. It is designed to run cleanly on GitHub Pages with no backend, no API integrations, and no external platform dependencies.

Tiny Cakes highlights:
- boutique bakery branding
- responsive layout design
- product category filtering
- quick-view product modal
- front-end cart drawer
- localStorage cart persistence
- gallery lightbox
- testimonial rotation
- scroll reveal animations

## Live Project Goals

The goal of this project was to transform a basic one-page class project into a stronger portfolio piece that feels closer to a real client-facing bakery website.

It was rebuilt to better demonstrate:
- UI design skills
- responsive front-end development
- DOM manipulation with JavaScript
- clean visual hierarchy
- stronger portfolio presentation for hiring

## Built With

- HTML5
- CSS3
- JavaScript

## Features

### Responsive Boutique Layout
The site is designed to adapt across desktop, tablet, and mobile screen sizes.

### Interactive Menu Filtering
Users can filter products by category:
- All
- Cupcakes
- Cookies
- Muffins
- Drinks

### Quick View Modal
Each product can be opened in a modal with additional detail and pricing.

### Front-End Cart Experience
Users can add items to a cart drawer, remove items, and see totals update dynamically.

### localStorage Support
Cart contents persist after page refresh using browser localStorage.

### Gallery Lightbox
Gallery images can be expanded in a lightbox overlay.

### Testimonial Slider
Testimonials rotate automatically to give the site more motion and polish.

### Scroll Reveal Animation
Sections animate into view as the user scrolls through the page.
